# 生产级登录注册系统指南

## 📱 功能概览

应用现在包含完整的生产级登录注册系统：

### 1. 手机号验证码登录
- 输入手机号
- 获取验证码（60秒倒计时）
- 输入验证码登录
- 自动注册新用户

### 2. 微信登录
- 一键微信授权登录
- 自动获取用户信息
- 快速完成注册

### 3. 测试账号登录（仅开发环境）
- 快速测试功能
- 无需真实手机号
- 仅在 `__DEV__` 模式下可见

## 🚀 使用步骤

### 步骤 1: 在 Supabase 创建测试用户

在 Supabase SQL Editor 中执行 `create_test_user.sql`:

```bash
cd OvertimeIndexApp
# 打开 create_test_user.sql 文件
# 复制内容到 Supabase Dashboard > SQL Editor
# 点击 Run 执行
```

**重要**: 测试用户 ID 已更新为有效的 UUID 格式：`00000000-0000-0000-0000-000000000001`

### 步骤 2: 重启应用

```bash
cd OvertimeIndexApp
npx expo start --tunnel --clear
```

### 步骤 3: 选择登录方式

应用启动后会显示登录界面，有三种登录方式：

#### 方式 1: 手机号登录（生产环境）
1. 输入手机号
2. 点击"获取验证码"
3. 输入收到的验证码
4. 点击"登录"按钮

#### 方式 2: 微信登录（生产环境）
1. 点击"💬 微信登录"按钮
2. 授权微信登录
3. 自动完成注册

#### 方式 3: 测试账号登录（仅开发环境）
1. 滚动到底部
2. 点击"🧪 测试账号登录"按钮
3. 自动登录测试账号

## ⚠️ 注意事项

### 开发环境
- 测试账号登录按钮仅在开发环境（`__DEV__`）可见
- 生产环境不会显示测试账号选项

### 手机号验证码
- 需要在 Supabase Dashboard 中配置 OTP 服务
- 或集成第三方短信服务（阿里云、腾讯云等）

### 微信登录
- 需要在 Supabase Dashboard 中配置微信 OAuth
- 需要微信开放平台账号
- 需要配置回调 URL

### 数据安全
- 测试账号是共享的，数据会互相覆盖
- 生产环境请使用真实的认证服务
- 敏感信息请加密存储

## 🐛 常见问题

### Q: UUID 错误 "invalid input syntax for type uuid"
A: 已修复。测试用户 ID 现在使用有效的 UUID 格式：`00000000-0000-0000-0000-000000000001`

### Q: 点击登录按钮没有反应？
A: 检查：
- 控制台日志
- 网络连接
- Supabase 配置
- Redux store 状态

### Q: 验证码发送失败？
A: 需要配置 Supabase OTP 服务或第三方短信服务

### Q: 微信登录提示"开发中"？
A: 需要在 Supabase 中配置微信 OAuth，并实现深度链接回调

### Q: 如何退出登录？
A: 进入设置页面，点击"退出登录"按钮
