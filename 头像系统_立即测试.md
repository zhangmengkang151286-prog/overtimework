## 🎨 SVG 文件要求

### 必须满足
- ✅ 文件格式：`.svg`
- ✅ 文件命名：`avatar_01.svg` 到 `avatar_20.svg`
- ✅ viewBox：正方形（如 `viewBox="0 0 100 100"`）

### 推荐规格
- 尺寸：100x100 或 200x200 像素
- 文件大小：< 20KB
- 颜色：纯色或简单渐变
- 避免：复杂滤镜、JavaScript、外部样式表

## 🐛 常见问题

### Q: 占位符头像不显示？
**A**: 清除缓存重启：`npx expo start --clear`

### Q: 添加 SVG 后报错？
**A**: 
1. 检查文件命名是否正确
2. 检查 SVG 的 viewBox 是否为正方形
3. 使用 SVGOMG 优化 SVG：https://jakearchibald.github.io/svgomg/

### Q: TypeScript 报错？
**A**: 
1. 确认 `src/types/svg.d.ts` 文件存在
2. 重启 VS Code
3. 运行 `npx tsc --noEmit` 检查错误
