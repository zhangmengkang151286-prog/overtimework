## 📱 测试步骤

### 1. 注册流程
1. 打开应用
2. 注册新账号
3. 进入"完善个人资料"页面
4. **查看头像选择器**：
   - 显示 20 个头像（4 列 x 5 行）
   - 只显示头像图标，无文字
   - 点击选择，白色边框高亮
5. 选择一个头像
6. 完成注册

### 2. 设置页面
1. 进入设置
2. 点击"编辑个人资料"
3. 查看头像选择器
4. 更换头像
5. 保存

### 3. 主页显示
1. 返回主页
2. 查看右上角头像
3. 确认显示正确

- **头像选择器**：4 列网格，只显示头像图标

## 🐛 如果遇到问题

### 显示数字占位符？
```bash
# 清除缓存重启
npx expo start --clear
```

### TypeScript 报错？
1. 重启 VS Code
2. 运行 `npx tsc --noEmit`

### SVG 不显示？
1. 确认 20 个 SVG 文件都在 `assets/avatars/` 目录
2. 清除缓存重启
