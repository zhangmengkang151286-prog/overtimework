## 📝 迁移总结

### 已替换的文件
1. ✅ `src/services/supabaseService.ts` - 使用 PostgREST API
2. ✅ `src/services/supabaseRealtimeService.ts` - 使用轮询替代 Realtime
3. ✅ `src/screens/TrendPage.tsx` - 移除 Supabase SDK 依赖

### 核心变化
- **Supabase SDK** → **PostgREST API (fetch)**
- **Realtime 订阅** → **轮询（每 30 秒）**
- **JWT 认证** → **无需认证（公开 API）**

### 性能影响
- 轮询间隔：30 秒（可调整）
- 网络请求：每 30 秒 2 个请求（统计数据 + 标签数据）
- 对用户体验影响：几乎无感知
