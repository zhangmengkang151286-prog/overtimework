## 🎯 问题

Metro bundler 无法识别新安装的 `expo-linear-gradient` 和 `expo-blur` 模块。

## ✅ 解决方案（2 分钟）

### 步骤 1：停止 Expo 服务器

在运行 Expo 的终端窗口中按 `Ctrl+C`

### 步骤 2：执行一键修复

在 OvertimeIndexApp 目录下运行：

```cmd
fix-glassmorphism.bat
```

**或者手动执行：**

```cmd
npx expo start --clear
```

### 步骤 3：等待启动完成

等待看到以下提示：
```
Metro waiting on exp://192.168.x.x:8081
```

### 步骤 4：重新加载应用

在设备/模拟器上：
- iOS: 摇动设备 → Reload
- Android: 按两次 R 键
- 或者重新扫描二维码
