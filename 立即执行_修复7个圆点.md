## 如果遇到问题

### 问题1：Supabase 执行 SQL 报错
- 确保你有管理员权限
- 检查 SQL 语法是否完整复制

### 问题2：GitHub Actions 运行失败
- 检查 GitHub Secrets 是否配置正确：
  - `SUPABASE_URL`
  - `SUPABASE_ANON_KEY`
- 查看 Actions 日志中的错误信息

### 问题3：应用中圆点还是显示0
1. 确认 Supabase 中 `daily_history` 表有数据：
   ```sql
   SELECT * FROM daily_history ORDER BY date DESC LIMIT 7;
   ```
2. 重新加载应用（摇一摇 → Reload）
3. 检查应用日志（Metro 控制台）
