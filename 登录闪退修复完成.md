# 登录闪退问题修复完成

## 问题描述

1. **获取验证码时闪退**：点击"获取验证码"按钮时应用崩溃
2. **密码登录后闪退**：使用密码登录成功后应用崩溃

## 问题原因

### 1. 获取验证码闪退

**根本原因**：`AliyunSMSProvider` 构造函数中直接抛出异常

```typescript
// 错误代码
constructor() {
  // ...
  if (!this.signName) {
    throw new Error('ALIYUN_SMS_SIGN_NAME is not configured');
  }
  if (!this.accessKeyId || !this.accessKeySecret) {
    throw new Error('Aliyun credentials are not configured');
  }
}
```

当环境变量未正确加载时，构造函数抛出异常导致应用崩溃。

### 2. 密码登录后闪退

**根本原因**：Redux 不支持存储非序列化数据（Date 对象）

```typescript
// 错误代码
const userWithAvatar = {
  // ...
  createdAt: new Date(result.user.createdAt),  // ❌ Date 对象
  updatedAt: new Date(result.user.updatedAt),  // ❌ Date 对象
};
dispatch(setUser(userWithAvatar));
```

Redux 要求所有存储的数据必须是可序列化的，Date 对象会导致应用崩溃。

#### 创建辅助函数（LoginScreen.tsx）

```typescript
// 准备用户数据用于 Redux（确保所有数据都是可序列化的）
const prepareUserForRedux = (user: any) => {
  return {
    ...user,
    avatar: user.avatarUrl || '',
    phoneNumber: user.phoneNumber,
    province: user.province || '',
    city: user.city || '',
    industry: user.industry || '',
    company: user.company || '',
    position: user.position || '',
    workStartTime: user.workStartTime || '09:00',
    workEndTime: user.workEndTime || '18:00',
    // 保持日期为字符串格式，Redux 不支持 Date 对象
    createdAt: typeof user.createdAt === 'string' 
      ? user.createdAt 
      : user.createdAt?.toISOString?.() || new Date().toISOString(),
    updatedAt: typeof user.updatedAt === 'string' 
      ? user.updatedAt 
      : user.updatedAt?.toISOString?.() || new Date().toISOString(),
  };
};
```

## 技术要点

### Redux 序列化规则

Redux 要求所有存储的数据必须是可序列化的：

✅ **允许的类型**：
- 字符串（string）
- 数字（number）
- 布尔值（boolean）
- null
- 数组（Array）
- 普通对象（Plain Object）

❌ **不允许的类型**：
- Date 对象
- Function
- Map / Set
- Symbol
- 类实例

### 日期处理最佳实践

1. **存储**：使用 ISO 字符串格式
   ```typescript
   createdAt: '2024-03-03T10:30:00.000Z'
   ```

2. **显示**：需要时转换为 Date 对象
   ```typescript
   const date = new Date(user.createdAt);
   const formatted = date.toLocaleDateString();
   ```

3. **比较**：可以直接比较 ISO 字符串
   ```typescript
   if (user.createdAt > user.updatedAt) {
     // ...
   }
   ```
