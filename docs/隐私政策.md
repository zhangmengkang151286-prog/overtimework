# 下班指数 隐私政策

最后更新日期：2026年3月8日

生效日期：2026年3月8日

上海淘客翁技术服务有限公司（以下简称"我们"）深知个人信息对您的重要性，我们将按照法律法规的规定，保护您的个人信息及隐私安全。我们制定本《隐私政策》并特别提示：希望您在使用「下班指数」应用程序（以下简称"本应用"）前仔细阅读并充分理解本政策。

## 一、我们收集的信息

### 1.1 您主动提供的信息

| 信息类型 | 具体内容 | 收集目的 |
|---------|---------|---------|
| 账号信息 | 手机号码 | 注册登录、身份验证、发送短信验证码 |
| 个人资料 | 用户名、头像（应用内置头像选择） | 个人主页展示 |
| 地区信息 | 省份、城市 | 地区维度的数据统计分析 |
| 职业信息 | 行业、职位分类、具体职位 | 行业维度的数据统计分析 |
| 工作时间 | 上班时间、下班时间 | 计算加班状态的参考基准 |
| 工作状态 | 每日加班/准时下班状态、原因标签 | 核心功能：状态记录与统计 |
| 安全信息 | 登录密码（可选设置） | 账号安全，密码经加密存储 |

### 1.2 我们自动收集的信息

- 设备信息：设备型号、操作系统版本（用于适配和问题排查）
- 应用使用数据：应用启动时间、页面访问记录（用于改善用户体验）

### 1.3 我们不收集的信息

- 我们不收集您的精确地理位置（GPS）
- 我们不收集您的通讯录、通话记录
- 我们不收集您的相册内容
- 我们不收集您的雇主名称、薪资等敏感劳动信息
- 我们不使用任何第三方广告追踪 SDK

## 二、我们如何使用您的信息

2.1 提供核心服务：
- 使用手机号和验证码完成注册登录
- 记录和展示您的每日工作状态
- 生成个人月历视图和长期趋势分析
- 生成匿名聚合的群体趋势统计数据

2.2 改善服务体验：
- 分析应用使用情况以优化功能
- 排查和修复技术问题

2.3 安全保障：
- 验证用户身份，防止恶意注册
- 保障账号和数据安全

## 三、数据的匿名化处理

3.1 趋势页面展示的所有群体数据均为匿名聚合统计结果，包括：
- 加班/准时下班人数比例
- 加班/准时下班标签分布

3.2 任何用户均无法通过群体统计数据识别到特定个人。

3.3 您的个人工作状态记录仅在"我的"页面对您本人可见。

## 四、信息存储与安全

4.1 存储位置：您的数据存储在位于中华人民共和国境内的阿里云服务器上。

4.2 存储期限：
- 账号存续期间，我们持续保存您的数据
- 账号注销后，我们将在30个自然日内删除您的个人数据
- 匿名化处理后的聚合统计数据不在删除范围内

4.3 安全措施：
- 用户密码使用 SHA256 加密算法存储，我们无法获知您的明文密码
- 数据传输采用加密通道
- 数据库访问实施严格的权限控制
- 定期进行安全审计和漏洞排查

## 五、信息共享与披露

5.1 我们不会向任何第三方出售您的个人信息。

5.2 我们不会与任何第三方共享可识别您个人身份的信息，以下情况除外：
- 获得您的明确同意
- 根据法律法规的要求、司法机关或行政机关的强制性要求
- 为保护我们、用户或公众的权益、财产或安全所必需

5.3 第三方服务说明：
- 短信验证码服务：由阿里云短信服务提供，仅传输手机号码用于发送验证码
- 我们不集成任何第三方广告、社交分析或用户画像 SDK

## 六、您的权利

根据相关法律法规，您享有以下权利：

6.1 查阅权：您可以在"设置"页面查看和修改您的个人资料信息。

6.2 更正权：您可以随时修改您的用户名、头像、地区、行业、职位等信息。

6.3 删除权：您可以申请删除您的账号及相关个人数据。

6.4 撤回同意权：您可以通过系统设置关闭信息通知等权限授权。

6.5 注销账号：您可以通过应用内设置页面申请注销账号。注销后，我们将删除您的个人信息（法律法规另有规定的除外）。

## 七、设备权限使用说明

本应用可能申请以下设备权限：

| 权限 | 用途 | 是否必须 |
|-----|------|---------|
| 网络访问 | 数据同步和API通信 | 是 |
| 通知推送 | 提醒用户提交每日工作状态 | 否 |

您可以在系统设置中随时关闭非必要权限，不影响应用核心功能的使用。

## 八、未成年人保护

8.1 本应用主要面向具有劳动能力的成年用户。

8.2 我们不会故意收集未满14周岁未成年人的个人信息。如果我们发现在未获得可验证的监护人同意的情况下收集了未成年人的个人信息，我们将尽快删除相关数据。

8.3 如果您是未成年人的监护人，当您对所监护的未成年人的个人信息有相关疑问时，请通过本政策中的联系方式与我们联系。

## 九、隐私政策的更新

9.1 我们可能会不时更新本隐私政策。更新后的政策将在本应用内公布。

9.2 对于重大变更，我们将通过应用内通知等方式提醒您。

9.3 继续使用本应用即表示您同意更新后的隐私政策。

本隐私政策的解释和执行均适用中华人民共和国法律，包括但不限于《中华人民共和国个人信息保护法》《中华人民共和国网络安全法》《中华人民共和国数据安全法》。
