## 已知问题详情

### 1. Tamagui 测试环境兼容性

**问题**: Tamagui v2 与 Jest/react-test-renderer 存在兼容性问题

**影响**: 
- 部分单元测试失败
- 不影响应用实际运行

**解决方案**:
- 使用手动测试验证功能
- 考虑使用 E2E 测试
- 等待 Tamagui 团队修复

### 2. Supabase 类型定义

**问题**: Supabase 客户端返回 `never` 类型

**影响**:
- TypeScript 类型检查报错
- 不影响应用实际运行

**解决方案**:
- 重新生成 Supabase 类型定义
- 或使用类型断言

### 3. User 类型不匹配

**问题**: `enhanced-auth` 的 User 类型缺少 `avatar` 字段

**影响**:
- TypeScript 类型检查报错
- 不影响应用实际运行

**解决方案**:
- 统一两个 User 类型定义
- 或添加缺失字段

### 4. 代码格式化

**问题**: 存在 Prettier 格式化问题

**影响**:
- ESLint 检查报错
- 不影响代码功能

**解决方案**:
```bash
npm run format
```
