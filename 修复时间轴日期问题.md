# 修复时间轴日期问题

## 🐛 问题分析

**日志显示**：
```
LOG  [HourlySnapshot] Fetching snapshot from database: date=2026-01-30, hour=7
```

**实际情况**：
- 当前时间：2026-01-31 23:45（今天晚上11点45分）
- 数据库快照：2026-01-31（今天）
- 查询日期：2026-01-30（昨天）❌

**问题原因**：
时间轴的 `minTime` 和 `maxTime` 计算逻辑有问题。当前时间是 23:45，属于今天的工作日（06:00 - 次日05:59），但时间轴可能把它当成了昨天的工作日。

---

## 🔍 根本原因

在 `TrendPage.tsx` 的 `getTimeAxisRange` 函数中：

```typescript
const getTimeAxisRange = () => {
  const now = new Date();
  const currentHour = now.getHours();
  
  // 工作日开始时间（今天或昨天的 06:00）
  const workdayStart = new Date(now);
  if (currentHour < 6) {
    // 如果现在是 00:00-05:59，工作日从昨天 06:00 开始
    workdayStart.setDate(workdayStart.getDate() - 1);
  }
  workdayStart.setHours(6, 0, 0, 0);
  
  // ...
};
```

**问题**：
- 当前时间 23:45，`currentHour = 23`
- 不满足 `currentHour < 6`，所以 `workdayStart` 是今天 06:00 ✅
- 但是时间轴拖动到 7:00 时，传递的日期却是昨天！

这说明问题不在 `getTimeAxisRange`，而在**时间轴组件如何使用 `minTime`**。

---

## 💡 解决方案

问题在于：当用户拖动时间轴时，`TimeAxis` 组件使用 `minTime` 作为基准计算时间，但没有正确处理日期。

### 修复方法

在 `TimeAxis.tsx` 中，`positionToTime` 函数需要确保使用正确的日期：

```typescript
const positionToTime = (position: number): Date => {
  const clampedPosition = Math.max(0, Math.min(position, axisWidth));
  const timeOffset = (clampedPosition / axisWidth) * timeRange;
  const calculatedTime = new Date(minTime.getTime() + timeOffset);
  
  // 限制不能超过当前时间
  const now = new Date();
  if (calculatedTime > now) {
    return now;
  }
  
  return calculatedTime;
};
```

这个函数是正确的，它基于 `minTime` 计算时间。

**真正的问题**：`minTime` 的日期是正确的，但是当时间跨越午夜时（比如从 23:00 到 01:00），日期会自动调整。

## 🔧 修复代码

问题在于 `getTimeAxisRange` 函数的逻辑。当前时间是 23:45，应该使用**今天 06:00** 作为起点，但实际上可能使用了**昨天 06:00**。

让我检查一下...实际上代码逻辑是对的：
- 当前时间 23:45
- `currentHour = 23`
- 不满足 `currentHour < 6`
- 所以 `workdayStart` 应该是今天 06:00

**但是**，可能是因为 `workdayStart` 对象被修改了！

### 正确的修复

```typescript
const getTimeAxisRange = () => {
  const now = new Date();
  const currentHour = now.getHours();
  
  // 工作日开始时间（今天或昨天的 06:00）
  let workdayStart: Date;
  if (currentHour < 6) {
    // 如果现在是 00:00-05:59，工作日从昨天 06:00 开始
    workdayStart = new Date(now);
    workdayStart.setDate(workdayStart.getDate() - 1);
    workdayStart.setHours(6, 0, 0, 0);
  } else {
    // 如果现在是 06:00-23:59，工作日从今天 06:00 开始
    workdayStart = new Date(now);
    workdayStart.setHours(6, 0, 0, 0);
  }
  
  // 工作日理论结束时间（明天或今天的 05:59）
  const workdayTheoryEnd = new Date(workdayStart);
  workdayTheoryEnd.setDate(workdayTheoryEnd.getDate() + 1);
  workdayTheoryEnd.setHours(5, 59, 59, 999);
  
  // 实际结束时间：不超过当前时间
  const workdayEnd = now < workdayTheoryEnd ? now : workdayTheoryEnd;
  
  console.log('[TrendPage] Time axis range:', {
    now: now.toISOString(),
    workdayStart: workdayStart.toISOString(),
    workdayEnd: workdayEnd.toISOString(),
  });
  
  return {minTime: workdayStart, maxTime: workdayEnd};
};
```

添加日志后，我们可以看到 `minTime` 的实际值。
