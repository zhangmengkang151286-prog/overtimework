## 📊 技术实现

### 数据流
```
用户选择头像
  ↓
AvatarPicker 组件
  ↓
onSelect(avatarId)
  ↓
保存到状态 (selectedAvatar)
  ↓
提交到数据库 (users.avatar_url)
  ↓
Avatar 组件渲染
```

### 组件使用

**显示头像**：
```typescript
import {Avatar} from '../data/builtInAvatars';

<Avatar avatarId={user.avatar} size={44} />
```

**选择头像**：
```typescript
import {AvatarPicker} from '../components/AvatarPicker';

<AvatarPicker
  selectedId={selectedAvatar}
  onSelect={setSelectedAvatar}
/>
```

### 数据库字段
- 表：`users`
- 字段：`avatar_url`
- 类型：`TEXT`
- 存储值：`avatar_01` 到 `avatar_20`
