# 快速修复：7个圆点显示问题

## 问题
7个圆点显示数据都是0（除了1月30日）

## 原因
没有自动归档服务在运行

## 解决方案（3步，10分钟）

### 📝 第1步：Supabase 创建归档函数

1. 打开 Supabase Dashboard
2. 点击左侧 "SQL Editor"
3. 打开文件 `OvertimeIndexApp/setup_auto_daily_archive.sql`
4. 复制全部内容
5. 粘贴到 SQL Editor
6. 点击 "Run" 执行
7. ✅ 看到成功消息

### 📝 第2步：Supabase 手动归档最近7天

1. 还是在 SQL Editor
2. 打开文件 `OvertimeIndexApp/manual_archive_recent_days.sql`
3. 复制全部内容
4. 粘贴到 SQL Editor
5. 点击 "Run" 执行
6. ✅ 看到最近7天的数据

### 📝 第3步：推送自动归档工作流

```bash
# 在项目根目录执行
git add .
git commit -m "feat: 添加每日自动归档功能"
git push origin main
```

然后：
1. 打开 GitHub 仓库
2. 点击 "Actions" 标签页
3. 找到 "Daily Archive" 工作流
4. 点击 "Run workflow" 按钮
5. 点击绿色的 "Run workflow" 确认
6. ✅ 等待30秒，看到绿色勾

### 📱 第4步：验证

1. 手机上摇一摇
2. 点击 "Reload"
3. ✅ 看到7个圆点显示正确的颜色

## 完成！

从现在开始，每天早上6点会自动归档数据，无需再次操作。
