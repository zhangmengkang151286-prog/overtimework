## 🎯 解决方案（按顺序尝试）

### 方案 1：标准清除流程（推荐，90% 有效）

```bash
# 1. 停止 Metro（按 Ctrl+C）

# 2. 清除 Metro 缓存
cd OvertimeIndexApp
npm start -- --reset-cache

# 3. 在手机上完全卸载 APP
#    iOS: 长按图标 → 删除 APP
#    Android: 设置 → 应用 → 卸载

# 4. 重新扫码安装
```

**为什么要卸载 APP？**
- 只是重启 Metro 不够，因为手机上的 APP 还是旧版本
- 必须完全卸载，让 Metro 重新安装新版本

---

### 方案 2：深度清除（如果方案 1 不行）

```bash
# 1. 停止 Metro（Ctrl+C）

# 2. 删除所有缓存目录
cd OvertimeIndexApp
rmdir /s /q .expo
rmdir /s /q node_modules\.cache
del /q /s *.log

# 3. 清除 Metro 缓存并启动
npm start -- --reset-cache --clear

# 4. 完全卸载手机上的 APP

# 5. 重新安装
```

---

### 方案 3：完全重建（终极方案，100% 有效）

```bash
# 1. 停止所有进程

# 2. 完全删除依赖和缓存
cd OvertimeIndexApp
rmdir /s /q node_modules
rmdir /s /q .expo
rmdir /s /q .expo-shared
del package-lock.json

# 3. 重新安装依赖
npm install

# 4. 清除缓存启动
npm start -- --reset-cache --clear

# 5. 完全卸载手机 APP

# 6. 重新安装
```

## 📱 手机端操作详解

### iOS 设备

1. **完全卸载 APP**
   ```
   长按 APP 图标
   → 点击"删除 APP"
   → 选择"删除 APP"（不是"移到 App 资料库"）
   ```

2. **确认已删除**
   - 在主屏幕和 App 资料库都找不到该 APP

3. **重新安装**
   - 运行 `npm start`
   - 使用相机扫描二维码
   - 等待安装完成

### Android 设备

1. **完全卸载 APP**
   ```
   设置 → 应用 → Expo（或你的 APP 名称）
   → 卸载
   ```

2. **清除数据**（可选，更彻底）
   ```
   在卸载前：
   设置 → 应用 → Expo
   → 存储 → 清除数据
   → 然后卸载
   ```

3. **重新安装**
   - 运行 `npm start`
   - 扫描二维码
   - 等待安装完成

## 🚨 如果还是不行

### 检查清单

- [ ] 已停止 Metro（Ctrl+C）
- [ ] 已运行 `npm start -- --reset-cache`
- [ ] 已在手机上**完全卸载** APP（不是关闭）
- [ ] 已重新扫码安装
- [ ] 已打开 APP 查看效果

### 可能的其他原因

#### 1. 使用了 Development Build

如果你用的是 Development Build（不是 Expo Go），需要重新构建：

```bash
# 重新构建
eas build --profile development --platform ios
# 或
eas build --profile development --platform android
```

#### 2. 有多个 Metro 进程

检查是否有多个 Metro 在运行：

```bash
# Windows
tasklist | findstr node

# 如果看到多个 node.exe，全部结束：
taskkill /F /IM node.exe

# 然后重新启动
npm start -- --reset-cache
```

#### 3. 代码没有保存

确认所有文件都已保存：
- `src/theme/colors.ts`
- `src/theme/typography.ts`
- `src/store/slices/uiSlice.ts`

在 VS Code 中按 `Ctrl+K S` 保存所有文件。
