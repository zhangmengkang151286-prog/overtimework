# 头像系统升级 - 现代扁平化风格

## 设计灵感

参考 https://userpics.craftwork.design/ 的现代扁平化风格，重新设计了 20 个系统内置头像。

## 设计特点

### 1. 现代渐变背景
每个头像都有独特的双色渐变背景，使用 LinearGradient 实现：
- 紫色系：`#667eea → #764ba2`
- 粉色系：`#f093fb → #f5576c`
- 蓝色系：`#4facfe → #00f2fe`
- 绿色系：`#43e97b → #38f9d7`
- 等 20 种不同的渐变组合

### 2. 扁平化设计
- 简洁的几何形状
- 圆形脸部
- 简化的发型路径
- 极简的五官表现

### 3. 柔和的配色
- 多种肤色：从浅粉到小麦色
- 多样发色：黑色、棕色、金色、粉色等
- 现代衣服颜色：蓝色、紫色、红色、绿色等
- 所有颜色都使用渐变增强立体感

### 4. 友好的表情
- 微笑的嘴巴曲线
- 明亮的眼睛高光
- 可爱的腮红效果

### 5. 多样化配饰
- **眼镜**：适合知性、商务风格
- **耳环**：适合时尚、优雅风格
- **项链**：适合职场、精致风格
- **无配饰**：适合运动、休闲风格

## 技术实现

### SVG 渐变
使用 `react-native-svg` 的 `LinearGradient` 组件：
```typescript
<Defs>
  <LinearGradient id="bg-avatar_01" x1="0%" y1="0%" x2="100%" y2="100%">
    <Stop offset="0%" stopColor="#667eea" />
    <Stop offset="100%" stopColor="#764ba2" />
  </LinearGradient>
</Defs>
<Circle cx="50" cy="50" r="50" fill="url(#bg-avatar_01)" />
```

### 组件结构
```typescript
export interface BuiltInAvatar {
  id: string;
  label: string;
  bgGradient: [string, string];      // 背景渐变
  skin: string;                       // 肤色
  hair: string;                       // 头发基础色
  hairGradient?: [string, string];    // 头发渐变（可选）
  shirt: string;                      // 衣服基础色
  shirtGradient?: [string, string];   // 衣服渐变（可选）
  accessory?: 'glasses' | 'earrings' | 'necklace' | 'none';
}
```

### 使用方式
```typescript
// 在任何组件中使用
import {Avatar} from '../data/builtInAvatars';

<Avatar avatarId="avatar_01" size={44} />
<Avatar avatarId={user?.avatar} size={72} />
```
