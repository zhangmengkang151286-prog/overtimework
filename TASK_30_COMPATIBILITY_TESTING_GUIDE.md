## 常见问题和解决方案

### 问题 1: iOS 上组件显示异常

**可能原因**:
- SafeAreaView 配置问题
- 字体加载问题
- 样式兼容性问题

**解决方案**:
```typescript
// 确保使用 SafeAreaView
import {SafeAreaView} from 'react-native-safe-area-context';

<SafeAreaView style={{flex: 1}}>
  {/* 内容 */}
</SafeAreaView>
```

### 问题 2: Android 上按钮点击无响应

**可能原因**:
- TouchableOpacity 配置问题
- z-index 层级问题

**解决方案**:
```typescript
// 使用 Pressable 替代 TouchableOpacity
import {Pressable} from 'react-native';

<Pressable onPress={handlePress}>
  {/* 内容 */}
</Pressable>
```

### 问题 3: 深色模式颜色不正确

**可能原因**:
- 颜色 token 使用错误
- 主题配置问题

**解决方案**:
```typescript
// 使用 gluestack-ui 的颜色 tokens
<Box bg="$backgroundLight0" _dark={{bg: "$backgroundDark900"}}>
  {/* 内容 */}
</Box>
```

### 问题 4: 小屏幕上布局溢出

**可能原因**:
- 固定宽度/高度
- padding/margin 过大

**解决方案**:
```typescript
// 使用 flex 布局
<Box flex={1} p="$4">
  {/* 内容 */}
</Box>
```
