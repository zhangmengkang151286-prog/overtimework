## 📝 添加真实 SVG（可选）

### 快速步骤
1. 下载 20 个 SVG 头像
2. 重命名为 `avatar_01.svg` 到 `avatar_20.svg`
3. 放到 `assets/avatars/` 目录
4. 打开 `src/data/builtInAvatars.tsx`
5. 取消注释导入语句（第 13-32 行）
6. 替换 `BUILT_IN_AVATARS` 数组（第 60 行）
7. 删除 `PlaceholderAvatar` 组件
8. 运行 `npx expo start --clear`

### 推荐网站
- https://userpics.craftwork.design/ - 现代扁平化
- https://www.flaticon.com/ - 免费图标
- https://www.humaaans.com/ - 可定制插画

## 🔧 组件使用

### 显示头像
```typescript
import {Avatar} from '../data/builtInAvatars';

<Avatar avatarId={user.avatar} size={44} />
```

### 选择头像
```typescript
import {AvatarPicker} from '../components/AvatarPicker';

<AvatarPicker
  selectedId={selectedAvatar}
  onSelect={setSelectedAvatar}
/>
```

## 🐛 常见问题

### 头像不显示？
```bash
npx expo start --clear
```

### TypeScript 报错？
1. 确认 `src/types/svg.d.ts` 存在
2. 重启 VS Code

### SVG 导入失败？
1. 确认 `metro.config.js` 已更新
2. 确认文件命名正确
3. 清除缓存重启
