# ⚡ 快速解决方案 - 3 步搞定

## 问题

代码已经改好了，但手机上的 APP 还是旧样子。

## 原因

Metro 缓存了旧代码，手机上的 APP 还是旧版本。

---

## 🎯 解决方法（3 步）

### 第 1 步：停止 Metro

在运行 Metro 的终端按 `Ctrl+C`

### 第 2 步：清除缓存重启

```bash
cd OvertimeIndexApp
npm start -- --reset-cache
```

### 第 3 步：重装 APP

1. **在手机上完全卸载 APP**
   - iOS: 长按图标 → 删除 APP
   - Android: 设置 → 应用 → 卸载

2. **重新安装**
   - 扫描 Metro 的二维码
   - 等待安装完成

## 🚨 如果还不行

### 方案 A：深度清除

```bash
cd OvertimeIndexApp
rmdir /s /q .expo
npm start -- --reset-cache --clear
```

然后卸载并重装 APP。

### 方案 B：完全重建

```bash
cd OvertimeIndexApp
rmdir /s /q node_modules
rmdir /s /q .expo
del package-lock.json
npm install
npm start -- --reset-cache
```

然后卸载并重装 APP。

## 💡 关键点

**为什么必须卸载 APP？**

- Metro 缓存了编译后的 JS 代码
- 手机上的 APP 还在运行旧的 JS bundle
- 只有完全卸载重装，才能加载新代码

**不够的操作：**
- ❌ 只重启 Metro（手机上还是旧 APP）
- ❌ 只重启手机（APP 代码没变）
- ❌ 只在 APP 中 Reload（还是旧 bundle）

**正确的操作：**
- ✅ 清除 Metro 缓存
- ✅ 卸载手机上的 APP
- ✅ 重新安装
